# Garage 愿景宣言

## 我们为什么存在

AI Agent 的能力正在爆炸式增长，但一个 Agent 真正有价值的不是某次对话有多聪明，而是它能不能**记住你、跟着你、长在你身上**。

今天的现实是：你的 AI 经验散落在 ChatGPT 的对话历史里、Claude Code 的 session 里、各种工具的配置文件里。换一个工具、换一台机器、换一个项目，你的 Agent 就变成了一个陌生人。

Garage 存在就是为了解决这个问题。

## 我们要做什么

打造一个**随身携带、不断成长的 Agent 操作系统**。

它不是一个 SaaS 产品，不是一个框架，不是一个平台。它是你自己的一块地——你的 skills、你的知识、你的 agents、你的经验，全部种在这块地里，你走到哪它跟到哪。

## 终极形态

想象这样的场景：

你打开任何一个 Agent 宿主（Claude Code、Hermes、Cursor、或者明年才出现的某个新工具），挂上你的 Garage 目录。几秒后，这个 Agent 就变成了**你的 Agent**——它知道你的编码风格、记得你上个月的架构决策、能调用你积累的 50 个 skills、知道怎么帮你写博客。

这不是一个配置同步工具。这是一个**能力基座**——你的 Agent 的一切能力都生长在这里，不依赖任何单一宿主。

## 核心信念

1. **数据归你**：所有知识、经验、配置都在你的 git 仓库里。不锁在任何云端。
2. **宿主可换**：Claude Code 是今天的首选，不是永远的绑定。核心约定不依赖任何单一工具。
3. **渐进增强**：第一天就能用，用得越多它越强。不需要一次性配置完成。
4. **人机共生**：系统从你的工作中学习，但所有关键决策由你做。它是你的放大器，不是你的替代品。
5. **可传承**：你的 skills、知识、经验可以被分享、复用、传承。一个人的积累可以变成社区的基础。

## 我们不做什么

- 不做 SaaS 平台。Garage 是本地优先的。
- 不做 Agent 运行时。Garage 是能力层，运行时交给宿主。
- 不做通用 AI 框架。Garage 面向 solo creator，不做大而全的"什么都支持"。
- 不替用户做决策。系统提供建议和自动化，但关键选择权永远在用户手里。

## 一句话总结

**Garage 是你的 Agent 能力的家。你带着它走，能力不会丢。**
