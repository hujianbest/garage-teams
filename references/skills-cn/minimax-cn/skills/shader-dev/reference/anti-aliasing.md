# 抗锯齿详细参考

## 先决条件
- 了解屏幕空间导数（`dFdx`、`dFdy`、`fwidth`）
- 多次缓冲设置（用于 TAA）
- 基本信号处理概念

## 采样理论（奈奎斯特）

**奈奎斯特-香农定理**指出：要准确表示信号，采样率必须 ≥ 2× 最高频率。用着色器术语来说：
- 像素网格=采样率
- 程序细节/边缘清晰度=信号频率
- 当细节频率>像素频率→锯齿（摩尔纹、爬行边缘）

**解决方案**：提高采样率 (SSAA) 或降低信号频率（分析 AA、滤波）。

## SSAA 实施细节

### 抖动模式
- **网格**：`offset = vec2(m, n) / AA - 0.5` — 简单、均匀的覆盖
- **旋转网格 (RGSS)**：旋转位置的 4 个样本 — 近水平/垂直线的边缘覆盖更好
- **Halton 序列**：准随机低差异 — 高样本数的最佳覆盖范围

### 性能
AA=2（4 个样本）是实时 SDF 场景的实际限制。 AA=3（9 个样本）仅适用于离线/屏幕截图质量。

## SDF 分析 AA 深入探讨

### 为什么 `fwidth` 有效

`fwidth(d) = abs(dFdx(d)) + abs(dFdy(d))` 近似计算一个像素上 SDF 值的变化量。使用它作为 smoothstep 宽度：
- 无论缩放级别如何，边缘过渡都精确地跨越约 1 个像素
- 无需纹理采样——纯粹分析
- 适用于任何 SDF 形状

### 符号覆盖距离

对于像素中心处值为“d”的 2D SDF：```
coverage ≈ clamp(0.5 - d / fwidth(d), 0.0, 1.0)
```这将有符号距离映射到近似的像素覆盖范围，相当于像素覆盖范围上的盒式过滤器。

## 带邻域钳位的 TAA

完整的 TAA 管道：
1. **抖动**：每帧按 Halton(2,3) 序列偏移像素中心
2. **渲染**：抖动位置的完整场景→缓冲区A
3. **重新投影**：使用运动向量查找当前位置的前一帧像素
4. **Clamp**：将历史颜色限制为当前帧的 3×3 邻域的最小/最大（防止重影）
5. **混合**：`输出 = mix(当前，clampedHistory，0.9)`

### 邻域钳位```glsl
vec3 minCol = vec3(1e10), maxCol = vec3(-1e10);
for (int x = -1; x <= 1; x++)
for (int y = -1; y <= 1; y++) {
    vec3 s = texelFetch(currentBuffer, ivec2(fragCoord) + ivec2(x,y), 0).rgb;
    minCol = min(minCol, s);
    maxCol = max(maxCol, s);
}
vec3 clampedHistory = clamp(history, minCol, maxCol);
```## FXAA 算法演练

1. **亮度计算**：将5个样本（中心+NSEW）转换为亮度
2. **边缘检测**：`lumaRange = lumaMax - lumaMin` — 如果低于阈值则跳过
3. **边缘方向**：比较水平与垂直亮度梯度以确定边缘方向
4. **亚像素混合**：沿边缘方向以 1/3 和 2/3 偏移量采样
5. **质量**：简化版使用2个水龙头；完整的 FXAA 3.11 沿边缘使用多达 12 个抽头，以实现更好的端点检测
