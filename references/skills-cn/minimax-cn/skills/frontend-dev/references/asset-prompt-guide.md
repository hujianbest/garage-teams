# 快速工程指南

## 图片提示

- 构图要具体：“主题左对齐，右侧留有负空间，用于文本叠加”
- 指定照明：“柔和的工作室照明”、“黄金时段背光”、“平面漫射光”
- 包括风格修饰符：“编辑摄影”、“3D 渲染”、“平面矢量图”
- 添加技术规格：“4K分辨率、锐焦、浅景深”
- 对于网络资产：始终提及“干净的背景”、“网络优化”、“高对比度以提高可读性”
- **绝不**在图像提示中包含文本，除非明确要求 - AI 文本渲染不可靠

## 视频提示

- 在括号中使用 MiniMax 相机命令：“[推入]”、“[卡车向左]”、“[跟踪拍摄]”等。
- 描述场景、主题、灯光和情绪——API 默认自动优化提示
- 对于网页背景：保持6秒持续时间，添加“[静态镜头]”以保持稳定性
- 最多 2,000 个字符

## 音频/TTS

- 指定流派、节奏 (BPM)、情绪和乐器
- 对于背景音乐：“没有人声，适合背景，不分散注意力”
- 对于声音效果：对声音事件非常具体
- 对于 TTS：选择语音匹配内容语言和说话者性别

## 预设快捷键

|快捷方式 |规格|
|----------|------|
| `英雄` | 16:9 (1280x720) 图像、电影、文本安全空间 |
| `拇指` | 1:1 (1024x1024) 图像，居中主体 |
| `图标` | 1:1 (1024x1024)，平面风格，干净的背景 |
| `头像` | 1:1 (1024x1024)，纵向，圆形裁剪就绪 |
| `横幅` | 21:9 (1344x576)，OG/社交横幅 |
| `肖像` | 2:3 (832x1248)，垂直纵向 |
| `移动` | 9:16 (720x1280)，移动全屏 |
| `背景视频` | 768P、6s、『【静态镜头】』、MiniMax Hailuo-2.3 |
| `视频` | 768P、6s、MiniMax Hailuo-2.3、提示自动优化 |
| `视频高清` | 1080P、6s、极简海洛-2.3 |
| '背景音乐' | 30 年代背景音乐，无人声，可循环播放 |
| `sfx` |音效短，< 3s |
| `tts` |文本转语音、MiniMax HD、MP3 |
| `叙述` |富有表现力的旁白声音，MiniMax |
