## 调色板规则（强制）

### 严格遵守调色板

**仅使用提供的调色板。请勿创建或修改颜色。**

- 所有颜色必须来自用户提供的调色板
- 不要使用调色板之外的颜色
- 不要修改调色板颜色（亮度、饱和度、混合）
- **唯一例外**：使用 `transparency` 属性添加透明度 (0-100)```javascript
// Correct: Using palette colors
slide.addShape(pres.shapes.RECTANGLE, { fill: { color: theme.primary } });
slide.addText("Title", { color: theme.accent });

// Wrong: Colors outside palette
slide.addShape(pres.shapes.RECTANGLE, { fill: { color: "1a1a2e" } });
```### 无渐变

**禁止使用渐变。仅使用纯色。**

### 无动画

**禁止动画和过渡。** 所有幻灯片必须是静态的。

### 推荐的字体搭配

|标题字体 |正文字体|
|------------|------------|
|格鲁吉亚 |卡利布里 |
|宋体黑 |宋体 |

**选择有趣的字体配对** - 不要默认使用 Arial。选择具有个性的标题字体，并将其与干净的正文字体配对。

### 正文没有粗体

**纯文本正文和标题/图例文本不得使用粗体。**

- 正文段落、描述→正常体重
- 标题、图例、脚注 → 正常重量
- 仅对标题和标题保留粗体```javascript
// Correct
slide.addText("Main Title", { bold: true, fontSize: 36, fontFace: "Arial" });
slide.addText("Body text here.", { bold: false, fontSize: 14, fontFace: "Arial" });

// Wrong
slide.addText("Body text here.", { bold: true, fontSize: 14 });
```---

## 风格食谱

通过调整角半径（“rectRadius”）和间距，可以以 4 种不同的视觉风格呈现相同的设计。选择适合演示基调的风格配方。

> **单位注释**：PptxGenJS 使用英寸。幻灯片尺寸为 10" x 5.625" (LAYOUT_16x9)。

### 风格概述

|风格|拐角半径|间距|最适合 |
|--------|--------------|---------|----------|
| **锋利且紧凑** | 0 ~ 0.05" | 紧密 | 数据密集、表格、专业报告 |
| **柔软且平衡** | 0.08" ~ 0.12" |中等|企业、商业演示、一般用途 |
| **圆润宽敞** | 0.15" ~ 0.25" |轻松|产品介绍、营销、创意展示 |
| **药丸和空气** | 0.3" ~ 0.5" |打开|品牌展示、发布会、高级演示|
